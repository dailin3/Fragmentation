# 计算机基础

## 子领域

  - [[TCP]] — 传输控制协议，三次握手，流量控制
  - [[中断]] — 硬件中断，异常处理，系统调用
  - [[内存]] — 堆栈，虚拟内存，缓存一致性
  - [[内存管理]] — 分配，回收，内存泄漏，智能指针
  - [[内核]] — 操作系统内核，驱动，系统接口
  - [[指针]] — 指针运算，指针与数组，函数指针
  - [[操作系统]] — 进程，线程，内存管理，文件系统
  - [[数据库]] — SQL，索引，事务，数据库设计
  - [[数据结构]] — 数组，链表，树，图，哈希表
  - [[死锁]] — 死锁条件，预防，避免，检测
  - [[算法]] — 排序，搜索，动态规划，复杂度分析
  - [[网络]] — TCP/IP，HTTP，DNS，socket编程
